Features / An assistant that answers from real data

An assistant that answers from real data

Backend

"Catch me up on #general", "what tasks am I assigned", "create a high-priority task for Karim" — the assistant calls the same APIs the browser does, with your token, so it can only ever see or do what you could do yourself. No service account, no privilege escalation.

  • Real function-calling: it reads your actual unread messages, tasks and calendar — never invents an answer.
  • Runs through an OpenAI-compatible endpoint, so it isn't locked to one model provider.
